# Yap

Voice-first coding for Cursor. Build aloud, get roasted.

## Two modes

- **Build mode** — describe an app by voice, Yap writes the code into your workspace.
- **Review mode** — pick a famous dev (Linus, Carmack, DHH, Uncle Bob, ...), they review the open file out loud in their own voice, and apply edits live.

## Setup

1. Install the extension.
2. Open Settings, search "Yap", and set:
   - `yap.elevenlabsApiKey`
   - `yap.geminiApiKey`
3. Click the Yap mic icon in the Activity Bar.
